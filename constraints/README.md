## 加载优先级

1. **必读**：`content-preferences.md`（身份与风格）
2. **按需**：根据任务类型加载对应 1-2 个约束文档
3. **素材查阅**：需要具体内容时读取 `assets/references/` 中的对应PDF
4. **不要全读**：避免上下文膨胀，只加载与当前任务相关的约束
