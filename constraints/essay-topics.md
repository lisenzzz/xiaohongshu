# 作文题型分类与破题方法

## 一、三大母题（近十年全覆盖）

拿到题目先定位母题，立意方向不跑偏。

- **人与自我**：成长、认知、心态、选择
- **人与社会**：个体与群体、认可度、竞争与合作
- **人与科技/人文**：技术与人、传统与现代、工具与价值

## 二、审题立意流程

### 2.1 拆解材料，圈出关键词

逐字读，圈出名词、动词、转折词。问自己：材料在说什么？哪几个概念在打架？

### 2.2 找二元对立

90%的上海卷题目都藏着一对核心矛盾。例如：

- 时间密度 → 客观时间 vs 主观体验
- 钝感 → 敏感 vs 钝感
- 独立思考 → 独立判断 vs 社会互动

找不到矛盾？自己加一组。比如"凡事发生皆有利于我"背后是"积极心态 vs 盲目乐观"。

### 2.3 细化矛盾：给关键词加定语

不要只写"A vs B"，要写"在什么语境下的A vs 什么语境下的B"。例如：

- 独立思考 ≠ 拒绝一切他人，而是在信息过载的时代保持判断主权
- 与他人一起思考 ≠ 盲从，而是在尊重事实和逻辑的前提下主动寻求碰撞

### 2.4 确定立场：不是二选一，而是重新定义

高分行文不是"我选A，因为A好"，而是"A和B看似对立，实则统一于更高层次的C"。用公式：**不是……也不是……而是……**

### 2.5 开头公式（80-100字）

摘抄材料核心句 + 点出矛盾 + 开门见山亮出中心论点

示例：
> 俗话说"物以稀为贵"，如今电商让商品触手可及，算法让信息无限丰沛。当稀缺性被科技消解，物的价值靠什么定义？我认为，贵不是稀缺，而是我们主动投入的注意力与情感。

## 三、不同作文类型的处理

### 3.1 现象类（如"数字囤积""时间密度"）

- **重点**：分析原因（为什么会有这种现象）+ 方法建议
- **破题**：列举典型现象 → 挖掘本质（资源有限/注意力稀缺/认知偏差）
- **注意**：不要只停留在"考试煎熬"，要上升到生命质量、认知机制

### 3.2 观点类（如"凡事发生皆有利于我""认可度"）

- **重点**：辩证分析（先承认合理性，再指出局限性）+ 重新定义
- **破题**：直接回应观点，用"不是…而是…"给出自己的判断
- **注意**：不要只选边站，要给出更高层次的统一

### 3.3 比喻类（如"道都成赛道了""眼睛的睁开与闭上"）

- **重点**：概念界定（先翻译比喻）+ 破题
- **破题**：把比喻解码成白话（赛道→竞争性路径，步道→非功利性路径）
- **注意**：不翻译直接写，必跑题

### 3.4 概念类（如"钝感""时间密度""贵"）

- **重点**：重新定义概念 + 分析原因 + 方法建议
- **破题**：解构常识（物以稀为贵→在今天不成立）→ 给出新定义（贵不是稀缺，是注意力）
- **核心心法**：定义深度决定文章深度。从对立面、时代背景、动词化三个角度重新定义
