# 作文框架与结构规范

## 一、整体结构：层进式，6-8段

不是"总-分-总"的平铺，而是层层递进。

### 推荐段落分配（6段式常见）

| 段落 | 名称 | 核心任务 | 字数 |
|------|------|----------|------|
| 第1段 | 破题引入 | 摘抄材料+点矛盾+亮论点 | 80-100 |
| 第2段 | 正面论证 | 论证A的合理性，内部细分2-3层 | 150-180 |
| 第3段 | 反面论证 | 论证B的合理性，内部细分2-3层 | 150-180 |
| 第4段 | 辩证统一 | 提出更高层次的统一 | 150-180 |
| 第5段 | 方法/追问 | 可操作建议或再推一层 | 120-150 |
| 第6段 | 升华结尾 | 回扣，用名言/画面收束 | 60-80 |

- 概念界定和深层追问不是必写项，根据题目类型灵活增减
- 总段落可控制在6-8段

### 八段式扩展框架

1. 扣材料 + 引论点（第一段）
2. 概念界定（第二段）
3. 优点/合理性分析（第三段）
4. 缺点/局限性批判（第四段）
5. 原因深度分析（第五段）
6. 辩证思考/统一（第六段）
7. 方法建议（第七段）
8. 总结升华（第八段）

## 二、论证段内部结构（核心技巧）

每个150-180字的论证段，内部必须细分层次，不能写成一大坨。

### 方法一：递进式（层层深入）

- 第一层：事实/现象层面（发生了什么）
- 第二层：原因/逻辑层面（为什么会这样）
- 第三层：价值/意义层面（这意味着什么）

示例（论证"物质之贵的虚妄"）：
> 电商让商品触手可及，限量球鞋下月就过时，热门综艺明年无人记得。这是因为消费社会里，物的价值被符号绑架，鲍德里亚称之为符号消费。物质的贵转瞬即逝，因为它不扎根于人的真实需求，而扎根于营销制造的幻觉。

### 方法二：并列式（多角度支撑）

从不同维度（个人/社会/历史、物质/精神/情感）展开。

示例（论证"精神之贵的浮现"）：
> 从个人看，你愿意放下手机专注读两小时书，这段时间就是贵的。从社会看，付费自习室一小时几十块，买的不是座位，是主动对抗干扰的决心。从情感看，你花一周亲手做的礼物，比网上下单秒到的商品更珍贵。

### 原则

每个论证段必须有2-3个清晰的层次或角度，用"首先/其次/最后"或"从…看/从…看"等标识词明确划分。
