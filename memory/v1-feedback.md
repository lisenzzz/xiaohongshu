# 帖子审稿反馈与专项约束

汇总各帖子初稿的审稿反馈，后续生成帖子时必须遵守。

## 一、封面/标题

- 标题不能太长，必须一行放得下
- 副标题不要用数字（如"5种"），用更吸引人的表达

## 二、开头

- 禁用"扎心"这个词，太常见、太烂俗
- 不要编造数据（如"均分42-46分""一类卷不到10%"），来源存疑，阅卷老师看了会质疑

## 三、整体结构

- 不能5个点平铺罗列，要有递进感
- 推荐结构：从"症状"→"病因"→"药方"层层递进

## 四、避雷点深化要求

- **避雷1（随着...的发展）**：不能只说"这不好"，要解释学生为什么爱这样写（安全、不用思考的心理机制）
- **避雷2（排排坐）**：要给出更具体的递进逻辑框架，不能只说"要有递进"
- **避雷3（素材）**：不能只说"用理论代替"，必须给一个完整示范——把苏轼具体替换成什么理论、怎么嵌入论证
- **避雷4（结尾）**：展开要更充分，不能太短
- **避雷5（没有问号）**：和避雷1（开头）有重叠，都是"缺乏思辨"的表现，需要合并或替换为其他更独立的避雷点

## 五、结尾金句

- 金句前面要有铺垫，不能突兀出现

## 思辨金句帖子专项约束（来自 v1 审稿反馈）

### 一、标题

- 帖子标题和封面标题不能一模一样
- 帖子标题：`思辨金句背完这13句｜作文直接提10分`
- 封面大标题：`思辨作文金句 学完冲55+`
- 封面副标题：`8年高三老师私藏 阅卷老师一眼给高分`
- 标题必须出现"思辨作文"或"思辨"关键词

### 二、编号

- 用纯数字（1. 2. 3.），不用数字emoji（1⃣），显示不稳定
- 三组各自从1开始编号，不跨组连续

### 三、emoji

- 分组标题用不同emoji区分（📖/💡/⚡），不要全篇统一
- 正文开头用🔥，结尾用📌

### 四、用词

- "提分"不用"拉分"
