# 内容创作经验沉淀

## 一、素材选取经验

- **用户想要的是"共鸣榜"，不是"教学榜"**：用户要求的素材排名是基于学生最常用、最眼熟的素材进行锐评，目的是让学生产生"对对对就是这样"的共鸣感，而不是基于素材本身的学术价值排名。
- **12个角度比按单个素材排列更系统**：素材按类型角度分类（课内素材、哲学理论、社会学理论、心理学理论、网络理论、社会现象、文学文化、经济学、历史跨课本、心理学效应、管理学定律、经典典故），比逐个素材排列更有结构感和说服力。
- **五个等级命名要用学生黑话**：夯、顶级、人上人、NPC、拉完了——这些等级名称比"优/良/中/差"更能引起目标受众共鸣，也更符合小红书和公众号的网感。

## 二、用户沟通关键节点

- **第一稿偏差大是因为理解错了需求**：用户说"由夯到拉的锐评"，第一稿理解为"按素材使用水平分级+给出正确用法示范"，实际上用户要的是"按素材角度分类排名+阅卷老师真实反应的吐槽"。
- **用户纠正时要认真听核心差异**：用户说"不是素材多个用法，是把好用和不好用的素材列举15个排名"，核心信息是：1) 列举具体素材/角度 2) 排序而非教学 3) 共鸣感优先于正确性。
- **角度数量需要和用户确认**：用户给了6个角度，要求补充到12个，具体哪12个需要通过提问让用户选择，不要自行决定。

## 三、资料书植入方式

- **中植入最合适**：在排名过程中自然带出（如"这个理论我在X讲里拆过，今天不展开了"），在结尾轻轻提一次。不要每个梯队都提，会显得太刻意。
- **植入要像在聊天中随口提到**，不能像广告插入。

## 四、文案风格要点

- **云老师人设**：八年高三经验、四校背景，语气亲切但有碾压感，像朋友聊天吐槽但底子是专业判断。
- **活人感的关键**：段落长短不一、有短句吐槽也有长段分析、偶尔冒出一个口语化的表达（比如"怎么说呢""然后呢"）、不要每段都工工整整。
- **避免AI痕迹**：不使用过于工整的三连排比、不使用太对称的结构、开头不要太套路（"我一般笑笑不说话"这种已经烂了）。
- **不用双引号**：所有内容中用书名号或其他替代。
- **不要用破折号**：用逗号、句号或自然衔接。
