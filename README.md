# 小红书内容工作空间

## 工作空间目的

本工作空间用于辅助"上海思辨写作云老师"小红书账号的内容创作，具体目标：

1. **选题规划**：围绕高考作文、思辨写作等方向，持续产出选题灵感
2. **内容生成**：基于小红书风格约束和账号调性，快速生成帖子草稿
3. **质量管控**：通过约束规则确保每篇内容符合平台风格和账号定位
4. **素材积累**：沉淀范文、素材、参考资料，形成可复用的内容资产

## 目录结构

```
xiaohongshu/
├── constraints/        # 约束与规则
│   └── ...
├── memory/             # 经验沉淀与复盘
│   ├── content-creation-lessons.md  # 内容创作经验（素材选取、风格、植入等）
│   └── v1-feedback.md               # 各帖子审稿反馈与专项约束
├── templates/          # 帖子模板
├── posts/              # 帖子内容
│   ├── ideas/          # 选题灵感 & 大纲
│   ├── drafts/         # 草稿
│   └── published/      # 已发布成品
├── assets/             # 素材
│   ├── images/         # 图片素材
│   └── references/     # 参考资料、竞品分析
```

## 使用流程
1. 在 `posts/ideas/` 中记录选题灵感
2. 基于 `templates/` 模板在 `posts/drafts/` 中撰写草稿
3. 参考 `constraints/` 中的规则进行调整
4. 参考 `memory/` 中的经验教训，避免踩重复的坑
5. 定稿后移入 `posts/published/`
6. 每次创作结束后，将新的经验沉淀到 `memory/`
